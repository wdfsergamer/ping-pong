# Pong

Pong на C + SDL2. Меню, бот (3 сложности), VS FRIEND.

## Сборка

```
# MSYS2
pacman -S mingw-w64-ucrt-x86_64-SDL2
make && make run
```

## Управление

| Клавиша | Действие |
|---------|----------|
| W/S     | Игрок 1 |
| ↑/↓     | Игрок 2 (только VS FRIEND) |
| Пробел  | Выбрать / Пауза |
| ESC     | Назад / Выход |
| W/S или ↑/↓ | Выбор сложности |

## Правила

До 10 очков. Мяч ускоряется на 8% после каждого удара.

## Сложности бота

- **EASY** — 30% скорости, тупит по центру
- **NORMAL** — 55% скорости
- **HARD** — 85% скорости
